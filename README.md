# Свод зарплаты, НДФЛ и удержаний

## Что делает
- Загружает 4 Excel-файла:
  - `Аванс`
  - `Подсчет`
  - `Удержания` (опционально)
  - `Сотрудники/рестораны` (опционально)
- Ищет в листах колонки с ФИО и суммами, объединяет сотрудников по ФИО.
- Если сотрудник встречается несколько раз (в том числе премиальные ведомости), суммы складываются.
- Считает по сотруднику:
  - `ЗП с НДФЛ = (Аванс + Подсчет) / 0.87`
  - `НДФЛ аванса = Аванс / 0.87 - Аванс`
  - `НДФЛ подсчета = Подсчет / 0.87 - Подсчет`
  - `Общий НДФЛ = ЗП с НДФЛ - (Аванс + Подсчет)`
- Добавляет поле `Удержания`.
- Проставляет `Ресторан` (подразделение) из отдельного файла.
- Сортирует сотрудников по схеме `Ресторан -> ФИО`.
- Показывает отдельный блок сотрудников без определенного ресторана.
- Строит свод `Сумма удержаний по ресторанам`.
- Поддерживает множественный фильтр по ресторанам.
- Блоки работают автономно:
  - можно загрузить только `Аванс + Подсчет` и получить расчет НДФЛ;
  - можно загрузить только `Удержания + Сотрудники/рестораны` и получить пофамильную сводку по ресторанам и суммам удержаний.
- Выгружает результат в новый Excel:
  - лист `Свод`
  - лист `Удержания по ресторанам`

## Подсказки по структуре файлов
- Для `Аванса` и `Подсчета` нужны колонки с ФИО и суммой.
- Для файла `Удержаний` нужны колонки с ФИО и суммой удержания.
- Для файла `Сотрудники/рестораны` нужны колонки с ФИО и рестораном/подразделением.
- Поиск колонок выполняется автоматически по заголовкам (например, `ФИО`, `Сумма`, `Удержание`, `Ресторан`, `Подразделение`).

## Как запустить
1. Откройте `index.html` в браузере.
2. Выберите файлы.
3. Нажмите `Рассчитать`.
4. При необходимости выберите несколько ресторанов (Ctrl/Cmd + клик) и нажмите `Применить фильтр`.
5. Нажмите `Выгрузить в Excel`.
